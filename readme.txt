== Description ==
GutenWise is a gutenburg editor plugin that provide gutenburg blocks for content creators. This plugin gives Content creators to lots of free blocks collection to add and change content looks.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= Q. Who should use GutenWise? =
A. The GutenWise is a complete package of unique and creative blocks that help build beautiful pages and posts on a website. Therefore, it is certainly an asset for all those who love Gutenberg.

= Q. What are the requirements to use the GutenWise? =
A. You only need to have the latest version of WordPress on your website, to begin with. The GutenWise is basically an addon for the default WordPress editor. Therefore, the latest WordPress installation along with a theme should be enough, to begin with.
